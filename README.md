## Cài đặt và phát triển local

**Yêu cầu môi trường:**

- [Node.js](https://nodejs.org/): 18.0.0+
- [Bun](https://bun.sh/)

**Cách chạy:**

```shell
git clone <link-repo-của-bạn>
cd <tên-thư-mục-repo>
```

```shell
# Cài dependencies
bun install
```

```shell
# Chạy dev server
bun dev
```

```shell
# Build dự án
bun run build
```

```shell
# Xem trước bản build
bun preview
```

```shell
# Tạo bài viết mới
bun new-post
```
